数值分析第六周作业

Posted on 2021-04-19 Edited on 2021-05-21

1

记 \[ R_{2n+1}(x) = f(x) - H_{2n+1}(x), \] 则\(R_{2n+1}(x_j) = R_{2n+1}'(x_j) = 0, j = 0,1,\cdots,n.\)于是 \[ R_{2n+1}(x) = K(x)w_{n+1}^2(x). \] 定义 \[E(t)=R_{2n+1}(t)-K(x)w_{n+1}^2(t),\] 则\(E(t)\)以\(x_0,x_1,\cdots,x_{n},x\)为零点,于是\(E'(t)\)有\(x,\xi_i,x_i,i=0,1,\cdots,n\)共\(2n+3\)个零点,且互不重复.故存在\(\xi\in[\min_i x_i, \max_i x_i]\) \[ E^{(2n+2)}(\xi) = 0\Leftrightarrow K(x) = \frac{f^{(2n+2)}(\xi)}{(2n+2)!}\Leftrightarrow f(x) - H_{2n+1}(x) = \frac{f^{(2n+2)}(\xi)}{(2n+2)!}w_{n+1}^2(x). \]

P105, 5

\[\int_{c}^d f(x,y) \mathrm{~d}y = \frac{d-c}{6}(f(x,c)+4f(x,\frac{c+d}{2})+f(x,d)) - \frac{(d-c)^5}{2880}\frac{\partial^4f}{\partial y^4}(x,\eta),\quad \eta \in (c, d).\] 记\[g(x)=\frac{d-c}{6}(f(x,c)+4f(x,\frac{c+d}{2})+f(x,d)),\]则 \[\int_{a}^b \mathrm{~d}x\int_{c}^d f(x,y) \mathrm{~d}y \approx \frac{b-a}{6}(g(a)+4g(\frac{a+b}{2})+g(b)).\] 由Simpson公式的结果知余项为 \[ -\frac{(b-a)^5}{2880}g^{(4)}(\xi) - \frac{(d-c)^5}{2880}\frac{\partial^4f}{\partial y^4}(x,\eta)= -\frac{d-c}{6}\frac{\partial^4}{\partial x^4}(f(\xi,c)+4f(\xi,\frac{c+d}{2})+f(\xi,d)) - \frac{(d-c)^5}{2880}\frac{\partial^4f}{\partial y^4}(x,\eta), \quad\xi\in(a,b). \]

P105, 8

  1. (3.3.14) \[ \begin{aligned} \left|\int_{a}^{b} f(x) \mathrm{~d} x-M(h)\right| = \left|\sum_{i=0}^{n-1}\int_{x_i}^{x_{i+1}} f(x) -f(x_{i+1/2})\mathrm{~d} x\right| \leq \sum_{i=0}^{n-1} \frac{h^{3}}{24}|f''(\xi_i)|\leq \frac{h^{2}}{24} M_{2}(b-a). \end{aligned} \]

  2. (3.3.15) \[ \left|\int_{a}^{b} f(x) \mathrm{~d} x-T(h)\right| = \left|\sum_{i=0}^{n-1}\int_{x_i}^{x_{i+1}} f(x) -\frac{1}{2}(f(x_{i})+f(x_{i+1}))\mathrm{~d} x\right| \leq \sum_{i=0}^{n-1} \frac{h^{3}}{12}|f''(\xi_i)|\leq \frac{h^{2}}{12} M_{2}(b-a). \]

  3. (3.3.16) \[ \begin{aligned} \left|\int_{a}^{b} f(x) \mathrm{d} x-S(h)\right| &= \left|\sum_{i=0}^{n-1}\int_{x_i}^{x_{i+1}} f(x) -\frac{x_{i+1}-x_i}{6}(f(x_{i})+4f(\frac{x_i+x_{i+1}}{2})+f(x_{i+1}))\mathrm{~d} x\right|\\ &\leq \sum_{i=0}^{n-1} \frac{h^{5}}{2880}|f^{(4)}(\xi_i)| \leq \frac{h^{4}}{2880} M_{4}(b-a). \end{aligned} \]

Ｐ106, 10

  1. 否则,假设\(P(x)\)在\((a,b)\)只有\(k\)个实根\(x_1,x_2,\cdots,x_k,k < n\).则取\[ Q_{k}(x)=\left(x-x_{1}\right)\left(x-x_{2}\right) \cdots\left(x-x_{k}\right)\in \text{span}\{1,x,x^2,\cdots,x^k\}.\] 则设\(P(x)=R(x)Q_k(x)\),其中\(R(x)\)在\((a,b)\)上定号,于是 \[\int_a^b P(x)Q_k(x) \mathrm{~d}x = \int_a^b R(x)Q_k^2(x) \mathrm{~d}x \neq 0.\text{矛盾!}\]

  2. \(\forall f(x)\in \mathbb{P}_{2n-1}, f(x) = P(x)Q(x)+R(x),Q,R\in \mathbb{P}_{n-1}\),于是 \[\begin{aligned} \int_a^b f(x)\mathrm{~d} x = \int_a^b P(x)Q(x)\mathrm{~d} x + \int_a^b R(x)\mathrm{~d} x=\sum_{i=1}^{n}A_iR(x_i)= \sum_{i=1}^{n}A_if(x_i). \end{aligned}\]

P106, 11

由 \[ f(x)=\sum_{j=0}^{n} f\left(x_{j}\right) l_{j}(x)+\frac{f^{(n+1)}(\xi)}{(n+1) !} \prod_{j=0}^{n}\left(x-x_{j}\right) \] 知\(r_i\)可取为 \[ r_i = \int_a^b l_{i}(x) \mathrm{~d}x. \] 若存在不恒同于\(r_i\)的\(r_i'\)使得 \[ \sum_{i=0}^{n} r_{i}' P\left(x_{i}\right)=\int_{a}^{b} P(x) \mathrm{~d} x, \] 取\(P_i\)满足\(P_i(x_i)=1,P_i(x_j)=0,\forall j\neq i\)即知\(r_i\equiv r'_i\),唯一性得证.
